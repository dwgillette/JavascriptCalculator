## Javascript Calculator

A basic calculator built from scratch with Javascript. A good exercise in testing. It's difficult to account for every scenario and input combination until coming across them with rigorous testing.

## Screenshots
!['Screenshot Home page'](https://github.com/dwgillette/JavascriptCalculator/blob/master/CalcScreenShot.png)

## Dependencies
- React
- React-DOM
- React-scripts
- Node-sass

## Getting Started
Start by running ```$ npm install``` inside the main project folder

Start up the node server using the command: ```$ npm start```

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
